# CPP Monorepo

Collection of C++ projects that share common modules and libraries.

# Development

## Dependencies

**General:**

- `$ brew install cmake`

**GUI:**

- `$ brew install sdl2`
- `$ brew install sdl2_image`
- `$ brew install sdl2_ttf`
- `$ brew install sdl2_mixer`

**Testing:**

- `$ brew install googletest`
